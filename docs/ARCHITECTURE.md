# Architecture: 01-profile-readme

## Overview
A high-performance, accessible developer profile built using modern React and Vite. The application follows a single-page architecture (SPA) with a focus on visual fidelity and micro-interactions.

## Tech Stack
-   **Framework**: React 18
-   **Styling**: Tailwind CSS v4
-   **Build Tool**: Vite 6
-   **Animations**: Framer Motion & GSAP
-   **Icons**: Lucide React

## Key Components
-   **Hero**: Main introduction with glassmorphism profile card.
-   **Skills**: Dynamic grid displaying core technical competencies.
-   **Stats**: Integration for live GitHub statistics and language usage.
-   **Theme Controller**: Context-aware light/dark mode persistence.

## Performance Optimization
-   Vite-driven HMR and lightning-fast builds.
-   Hardware-accelerated animations using GSAP and Framer Motion.
-   Optimized assets for near-instant load times.
