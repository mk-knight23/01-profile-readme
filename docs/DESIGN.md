# Design System: 01-profile-readme

## Visual Philosophy
The design follows a **Premium Glassmorphism** aesthetic, emphasizing depth, transparency, and sophisticated lighting.

## Color Palette
-   **Dark Mode**: Deep slates, semi-transparent whites, and vibrant accent glows.
-   **Light Mode**: Soft grays, crisp glass surfaces, and refined shadows.

## UI Elements
-   **Glass Cards**: Multi-layered transparency with backdrop-blur effects.
-   **Typography**: Immersive, high-fidelity font pairings for maximum readability.
-   **Interactive States**: Smooth transitions and hover feedback on all clickable elements.

## Motion Design
-   **Entrance Animations**: GSAP-driven staggered entry for UI elements.
-   **Micro-interactions**: Framer Motion for button hovers and card flips.
-   **Theme Transitions**: Seamless fade between light and dark modes.
