Design Decisions

Tradeoffs and technical choices

Architecture

Vanilla over Frameworks

Chose vanilla HTML/CSS/JS for this portfolio to demonstrate fundamentals and keep the site lightweight. A React portfolio would add 200KB+ for little gain on a mostly-static site.

Tradeoff: Manual DOM manipulation vs declarative components. Worth it for a single-page portfolio.
Design

No Dark Mode

Deliberately avoided dark mode. Developer portfolios over-index on dark themes. The print-like, editorial design stands out and reflects the content-first philosophy I bring to documentation and developer tools.

Tradeoff: Some users prefer dark mode. Accessibility considerations are met through high contrast.
Documentation

Monorepo for VIBE

Planning a monorepo structure for the VIBE ecosystem to share types, utilities, and LLM provider integrations across CLI, VS Code extension, and web components.

Tradeoff: Increased build complexity vs better code sharing and unified versioning.
