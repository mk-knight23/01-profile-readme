# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-21

### Added
- **SEO**: Comprehensive meta tags, OpenGraph, Twitter Cards
- **Structured Data**: JSON-LD Person schema for rich snippets
- **Accessibility**: Skip link, ARIA labels, keyboard navigation
- **Theme Toggle**: Dark/light mode with system preference detection
- **Keyboard Shortcut**: Alt+T for theme toggle
- **Focus Styles**: Visible focus indicators for keyboard users
- **Reduced Motion**: Support for `prefers-reduced-motion`
- **Print Styles**: Clean print layout
- **Template Features**: Fork/Star buttons, clear customization points
- **Documentation**: GETTING_STARTED, FEATURES, STACK, DEPLOY, CHANGELOG
- **CI/CD**: Lint step in GitHub Actions workflow
- **ESLint**: Linting configuration

### Changed
- Upgraded to marketplace-grade template standards
- Improved responsive design for mobile
- Enhanced glassmorphism effects
- Better color contrast for accessibility
- Restructured CSS with custom properties

### Fixed
- Console.log statement removed
- Package.json description cleaned up

## [1.0.0] - 2026-01-15

### Added
- Initial profile template
- Glassmorphism design

- Responsive layout
